## CV Tracker

This is a free web app that allows you to track your job applications. It's 100% free and private - all data is saved in your browser.

[Click here to start using the app](https://patrykso.github.io/cv-tracker/)

Made as one-evening fun project to get familiar with Google's Antigravity and help few of my friends.

Developed mainly with React, Vite, Tailwind and Dexie.js.

## Running it locally

1. Navigate to the project directory and install the required packages:
   ```bash
   npm install
   ```
2. Run the server with:
   ```bash
   npm run dev
   ```
3. Open your local web address (usually `http://localhost:5173`)

## Features
- **Timeline** – track all offers you applied to and their status with option to add extra information like job posting url or your sent cv.
- **Statistics** – track statistics of your job hunt and generate banners to share your journey
- **Custom statuses** – change colors and names of default application statuses, add custom statuses
- **Data import and export** – for backups of your data
